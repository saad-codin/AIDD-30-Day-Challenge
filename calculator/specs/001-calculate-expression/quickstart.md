# Quickstart for Calculate Expression

## Prerequisites

- Python 3.11

## Running the calculator

1.  Navigate to the `src` directory.
2.  Run the calculator with a string expression as an argument:

    ```bash
    python -m calculator.main "2+2"
    ```

## Running tests

1.  Navigate to the root of the project.
2.  Run pytest:

    ```bash
    pytest
    ```
